HERE WE ARE GOING TO DESCRIPE SOME THEORITICAL INFORMATION ABOUT HTML:


1-WWW: Stands for World Wide Web ,is a collection of documents and other files. Most of the documents are made using Hypertext Markup Language (HTML).


2-HTML: stands for Hypertext Markup Language. It's used to define the structure or layout of a web page.


3-Markup: that tells the browser what that text should look like, Or how it's arranged.


4-Whitespace: all whitespaces get collapsed into a single space when your Html page is rendered.


5-Void elements: is the elements that don't have closing tags (ex: breakline element, image elements).


6-Block elements: Elements that do create a block like the invisible boxes around your paragraphex:(<p>,< div>,<video>,<ol>,<h1>).


7-Inline Elements: Elements that don't create a block ex:(<br>,<strong>,<a>,<img>).


8-<li> element: this tag doesn't need to have a closing tag because it will never have a nested element inside it


9-URL: Stands for Uniform resource locator, it's an address that tells your browser how to locate a particular resource (a file for instance) in a standardized way (ex: Go to this lesson)

  • UNIFORM = Standardized
  • Resource = a file or other data object
  • Locator = address for finding things

8-<!DOCTYPE html>: tells the broser that this file is going to be treated as modern html, this tag prevent bugs when your file is enlarged with codes and avoid reaching quircks mode
